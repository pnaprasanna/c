RHEL 8 → RHEL 9 Upgrade Guide

Step-by-step migration process using Leapp

1. Install Required Packages

dnf install leapp-upgrade
dnf install leapp*
dnf install vdo

2. Network Configuration

Add the following entry to: /etc/sysconfig/network-scripts/ifcfg-eth0

TYPE=Ethernet 

systemctl start NetworkManager
systemctl enable NetworkManager

3. Migrate Network Configuration

nmcli connection reload
nmcli connection show
nmcli connection migrate eth0
nmcli -f NAME,UUID,TYPE,FILENAME connection show

4. Remove Teaming Packages

rpm -qa | grep -E 'teamd|libteam'
dnf remove teamd libteam
reboot

5. VDO Confirmation

leapp answer --section check_vdo.confirm=True

6. Pre-upgrade Validation

leapp preupgrade

7. Perform Upgrade

leapp upgrade --no-rhsm --iso /data/iso/rhel9.iso
