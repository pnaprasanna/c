## Python Commands Workaround for Portable Python

### To Install Modules
Navigate to your Portable Python directory:
```bash
C:\1DATA\Python3.10\Portable Python-3.10.5 x64\
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org duckduckai
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org paramiko
```
### To Upgrade pip
```
python -m pip install --upgrade pip --trusted-host pypi.org --trusted-host files.pythonhosted.org

```
